## Описание программы

*"Book Tracker" - это консольное приложение для ведения списка прочитанных книг. Позволяет управлять списком прочитанных книг с возможностью фильтрации по жанру и колличеству страниц  

---

## Требования для запуска
Что нужно установить на компьютер:
-Python 3.10 или выше
-Библиотеки: tkinter, os, json

## Как пользоваться
При запуске появляются строки для ввода:

-"Название:" (здесь нужно ввести название книги)
-"Автор:" (здесь нужно ввести имя автора)
-"Жанр:" (здесь нужно ввести жанр)
-"Страниц" (здесь нужно числом написать колличество страниц в книге)

После заполнения всех полей нажать кнопку "Добавить книгу" (все поля обязательно должны быть заполнены)

Добавленные книги будут отображаться внизу в виде таблицы 

Можно осуществлять сортировку по жанру книги и/или колличеству страниц. Для этого:

-в поле "Жанр:" вводим жанр книги
-в поле "Страниц от" вводим колличество страниц

После заполнения хотя-бы одного из полей будет выведен список книг с соответствующими параметрами. Если нажать на кнопку фильтрации когда поля не заполнены, фильтрация не будет применена и выведется обычный список со всеми книгами
